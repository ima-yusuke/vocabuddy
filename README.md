# VocaBuddy（ボキャバディ）

AI搭載の英語学習支援Webアプリケーション。単語の登録・管理、検索、テスト機能に加えて、Google Gemini AIを活用した返信アシスタント機能を備えています。

## 主な機能

### 📚 単語帳管理
- **AI自動補完**: 英単語の発音・品詞・意味・例文をAIが自動取得（NEW!）
- **文脈対応**: 文脈を入力することで、その文脈に合った意味を優先的に取得
- **単語の登録**: 英単語、日本語の意味（複数可）、例文を登録
- **単語一覧**: 登録した全ての単語を一覧表示（発音・品詞も表示）
- **検索機能**: 英単語または日本語の意味から検索
- **削除機能**: 不要な単語を削除（権限が必要）

### ✏️ 単語テスト
- ランダムに出題される4択クイズ
- 正解・不正解の判定と解説表示
- 例文を確認しながら学習可能

### 🤖 AI返信アシスタント（Google Gemini API使用）
- 友達からの英語メッセージに対する返信文を自動生成
- 単語帳の単語を自然に使った返信文を提案
- カジュアルで自然な会話表現を優先
- 英文と日本語訳、使用された単語を表示
- ワンクリックで英文をコピー

### 🔐 権限管理
- Laravel Permissionを使用した権限管理
- 「membership」権限を持つユーザーのみ単語の追加・削除が可能

## 技術スタック

- **バックエンド**: Laravel 11
- **フロントエンド**: Blade Templates, Tailwind CSS
- **データベース**: SQLite
- **認証**: Laravel Breeze
- **権限管理**: Spatie Laravel Permission
- **AI**: Google Gemini API (gemini-3-flash-preview)

## セットアップ

### 必要な環境

- PHP 8.2以上
- Composer
- Node.js & NPM

### インストール手順

1. **リポジトリをクローン**
```bash
git clone <repository-url>
cd vocabuddy
```

2. **依存関係をインストール**
```bash
composer install
npm install
```

3. **環境変数を設定**
```bash
cp .env.example .env
php artisan key:generate
```

4. **Gemini APIキーを設定**

`.env`ファイルに以下を追加：
```env
GEMINI_API_KEY=your_gemini_api_key_here
```

APIキーの取得方法：
1. [Google AI Studio](https://makersuite.google.com/app/apikey)にアクセス
2. 無料のAPIキーを作成
3. `.env`ファイルに貼り付け

5. **データベースを設定**
```bash
php artisan migrate
```

6. **権限を設定**
```bash
php artisan db:seed --class=RolesAndPermissionsSeeder
```
※ Seederファイルを作成して、membershipロールと権限を設定してください

7. **アセットをビルド**
```bash
npm run dev
```

8. **サーバーを起動**
```bash
php artisan serve
```

アプリケーションが `http://localhost:8000` で起動します。

## 使い方

### 単語の登録
1. 「membership」権限でログイン
2. トップページの「新しい単語を追加」フォームから登録
3. 英単語を入力し、「AIで意味を補完」をクリック（推奨）
   - AIが発音・品詞・意味・例文を自動取得
   - 文脈を入力すると、より適切な意味を取得可能
   - 補完結果は編集可能
4. または手動で英単語、意味、例文を入力
5. 「意味を追加」ボタンで複数の意味を登録可能

### 単語の検索
- 検索ボックスに英単語または日本語の意味を入力
- 部分一致で検索可能

### 単語テスト
1. 「単語テスト」ボタンをクリック
2. 表示された英単語の意味を4つの選択肢から選択
3. 結果と解説を確認
4. 「次の問題へ」で続けてテスト

### AI返信アシスタント
1. 「返信アシスタント」ボタンをクリック
2. 友達からの英語メッセージを入力
3. 返信したい内容を日本語で入力
4. 「返信文を生成」をクリック（10-30秒待機）
5. 生成された英文を確認し、コピーして使用

## 注意事項

- Gemini APIは無料枠内で使用できますが、大量のリクエストは制限される場合があります
- 返信アシスタント機能・自動補完機能は1回の生成に10-30秒程度かかります
- APIキーは絶対に公開リポジトリにコミットしないでください
